    Dogs & Cats - Image Recognition using CNN and Transfer Learning

    This notebook has the aim to tackle an image classification problem. The datasets were originally made available at a Kaggle competition, named "Dogs & Cats" dataset. The Dogs and Cats dataset contains of total 3000 images equally divided into cats and dogs images.

    To classify the images from this dataset, 4 different models will be trained by using the dogs & cats dataset.

    • Base Model
    • CNN Model
    • TransferLearning Model
    • Transfer Learning with Feature Extraction Model

    def plot_loss_curve(history):
  loss = history.history["loss"]
  val_loss = history.history["val_loss"]

  accuracy = history.history["accuracy"]
  val_accuracy = history.history["val_accuracy"]

  epochs = range(len(history.history["loss"]))

  # Plot loss
  plt.plot(epochs, loss, label="training_loss")
  plt.plot(epochs, val_loss, label="val_loss")
  plt.title("Loss")
  plt.xlabel("Epochs")
  plt.legend()

  # Plot accuracy
  plt.figure()
  plt.plot(epochs, accuracy, label="training_accuracy")
  plt.plot(epochs, val_accuracy, label="val_accuracy")
  plt.title("Accuracy")
  plt.xlabel("Epochs")
  plt.legend()

    def math_evaluate(evaluation_result):
  loss = tf.cast(evaluation_result[0], dtype=tf.float32)
  acc = tf.cast(evaluation_result[1], dtype=tf.float32)

  return {
      "loss": loss.numpy(),
      "acc": acc.numpy()
  }

    def create_model(model_url, num_class, image_shape, activation="softmax"):
  # Download pretrained model
  feature_extractor_layer = hub.KerasLayer(model_url,
                                           trainable=False,
                                           name="feature_ectraction_layer",
                                           input_shape=image_shape+(3,))

  model = tf.keras.Sequential([
                               feature_extractor_layer,
                               tf.keras.layers.Dense(num_class, activation=activation, name="output_layer")
  ])

  return model
