    Multiclass Image Classification with Fashion MNIST

    This code is the same as of TensorFlow tutorial and Coursera TensorFlow Specialization course. The code was run on a local machine to test how long and how much resource it would take to train a simple neural network model. The data visualization part and model evaluations aren't performed in this code.

    Highlights

    • Import data from TensorFlow Dataset
    • Use a single Callback to stop training the model once it reaches .99 accuracy level
    • Objective: To achieve an accuracy level of 0.99 without taking overfitting into account
    • Max epochs of 100
    • The final result hasn't met the expectation
    • It took all 100 epochs in approximately 12 minutes to train the model
    • Final result: - loss: 0.0409 - accuracy: 0.9850

    import tensorflow as tf

# Callback to stop training once reaches accuracy .99
class myCallback(tf.keras.callbacks.Callback):
    def on_epoch_end(self, epoch, logs={}):
        if (logs.get("accuracy") > .99):
            print("\nAccuracy has reached .99, stop training")
            self.model.stop_training = True

# Initialize callback
callbacks = myCallback()

# Load dataset from TensorFlow dataset
fashion_mnist = tf.keras.datasets.fashion_mnist

# Assign the dataset into train and test sets
(x_train, y_train), (x_test, y_test) = fashion_mnist.load_data()

# Resize
x_train, x_test = x_train / 255., x_test / 255.

# Create model
model = tf.keras.models.Sequential([
    tf.keras.layers.Flatten(input_shape=(28, 28)),
    tf.keras.layers.Dense(1024, activation="relu"),
    tf.keras.layers.Dense(10, activation="softmax")
])

# Compile model
model.compile(optimizer="adam",
              loss="sparse_categorical_crossentropy",
              metrics=["accuracy"])

# Run (fit) model
history = model.fit(
    x_train,
    y_train,
    epochs=100,
    callbacks=[callbacks]
)
    Final result: - loss: 0.0409 - accuracy: 0.9850
