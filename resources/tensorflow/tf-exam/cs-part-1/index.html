    Week 1: A New Programming Paradigm

    Week one is about introducing TensorFlow. The final exercise of this week is to predict house prices using TensorFlow instead of using typical mathematical methods.

    Exercise: Housing Price

    Problem

    • Train a simple regression model to predict housing price
    • The house price is 50k base price + 50k per each bedroom. Ex: house with 2 bedrooms cost 150k.
    • Predict the price of a house with 7 bedrooms

    Solution

    • Number of bedrooms xs = [1, 2, 3, 4, 5, 6]
    • Price in 100k ys = [1., 1.5, 2., 2.5, 3., 3.5]
    • The price prediction of a house with 7 bedrooms won't be exactly 4. It will show a slight variation since the mathematical error during each training cycle.

    import tensorflow as tf
import numpy as np

# Inputs
xs = np.array([1, 2, 3, 4, 5, 6], dtype=float)
ys = np.array([1., 1.5, 2., 2.5, 3., 3.5], dtype=float)

# Create model
model = tf.keras.Sequential([
          tf.keras.layers.Dense(units=1, input_shape=[1])
          ])

# Compile model
model.compile(optimizer="sgd", loss="mean_squared_error")

# Run model
model.fit(xs, ys, epochs=1000)

# Make prediction
print(model.predict([7.0])[0])

    The prediction result will be around 4.00XXXX with little variations from the third digits onwards

    Week 2: Introduction to Computer Vision

    This exercise it to train an image classification model and use a custom callback to stop the training once its accuracy reaches 99%. The dataset that is used to train the model is the Fashion MNIST dataset.

    From this lesson it is intended to use normal Dense layers to train the model. The end result shows an over-fitting problem, which means that the model hasn't found a pattern of data. In other words, it failed to generalize the data. This will be fixed by using convolutional layers in week's 3 lesson.

    Exercise: Handwriting Recognition

    Problem

    1. Should be successfully trained within 10 epochs
    2. When it reaches 99% or greater it should print out the string "Accuracy has reached .99, stop training"

    Solution

    • When train on a local machine (as of later on Google Colab), an accuracy of 99% couldn't be reached within 10 epochs. Only a 95% level of accuracy could be reached.
    • The time taken on Google Colab with T4 GPU vs MacBook Pro 2017 13" differs about 1 second per epoch. The later machine is slower.
    • Both machines can't reach an accuracy of 99% within 100 epochs. It took about 142 epochs to reach that level.

    import tensorflow as tf


class myCallback(tf.keras.callbacks.Callback):
    def on_epoch_end(self, epoch, logs={}):
        if (logs.get("accuracy") > .99):
            print("\nAccuracy has reached .99, stop training")
            self.model.stop_training = True


callbacks = myCallback()

fashion_mnist = tf.keras.datasets.fashion_mnist

(x_train, y_train), (x_test, y_test) = fashion_mnist.load_data()
x_train, x_test = x_train / 255., x_test / 255.

model = tf.keras.models.Sequential([
    tf.keras.layers.Flatten(input_shape=(28, 28)),
    tf.keras.layers.Dense(1024, activation="relu"),
    tf.keras.layers.Dense(10, activation="softmax")
])

# Compile model
model.compile(optimizer="adam",
              loss="sparse_categorical_crossentropy",
              metrics=["accuracy"])

# Run (fit) model
history = model.fit(
    x_train,
    y_train,
    epochs=100,
    callbacks=[callbacks]
)

    Week 4: Using Real-world Images

    To goal of this week is to multiple convolutional layers and the power of GPU to solve a binary-class image classification problem. The dataset consists of 80 images of happy and sad faces.

    Exercise: Handling Complex Images

    Problem

    1. When it reaches 99.9% or greater it should print out the string "Accuracy has reached .999, stop training"
    2. Use 3 Conv2D layers

    Solution

    • Run on Google Colab Tesla T4 VM
    • Preprocess image with ImageGenerater which is highly effective when using GPU
    • Create batches, dividing images into sets. This will be trained in sets instead of train on a single image per step.
    • In this model batch_size=40 of total images of 80. It can train 2 steps per epoch.
    • Set steps_per_epoch=2 in model.fit()
    • Time per epoch: less than 1 second
    • It took 19 epochs to reach an accuracy of 1.000 and loss 0.0662
    • In epoch 13 and 19, loss has swung between 0.0662 and 0.1858
    • Highly possible for over-fitting
    • NOTE Number of epochs, loss, and accuracy may vary in each run since each run the sequence of images changes randomly.

    import tensorflow as tf
from tensorflow.keras.optimizers import RMSprop
from tensorflow.keras.preprocessing.image import ImageDataGenerator
import os
import zipfile

# Download and extract dataset
!wget --no-check-certificate \
    "https://storage.googleapis.com/laurencemoroney-blog.appspot.com/happy-or-sad.zip" \
    -O "/tmp/happy-or-sad.zip"

zip_ref = zipfile.ZipFile("/tmp/happy-or-sad.zip", 'r')
zip_ref.extractall("/tmp/dataset_folder")
zip_ref.close()

# Prepare dataset
train_datagen = ImageDataGenerator(rescale=1/255)
validation_datagen = ImageDataGenerator(rescale=1/255)

train_generator = train_datagen.flow_from_directory("/tmp/dataset_folder",
                                                    target_size=(150, 150),
                                                    batch_size=40,
                                                    class_mode="binary")

# Setup callback
class myCallback(tf.keras.callbacks.Callback):
  def on_epoch_end(self, epoch, logs={}):
    if(logs.get("accuracy")>0.999):
      print("\nAccuracy has reached .999, stop training")
      self.model.stop_training = True

callbacks = myCallback()

# Create model
model = tf.keras.models.Sequential([
        tf.keras.layers.Conv2D(16, (3,3), activation="relu", input_shape=(150, 150, 3)),
        tf.keras.layers.MaxPooling2D(2, 2),
        tf.keras.layers.Conv2D(32, (3,3), activation="relu"),
        tf.keras.layers.MaxPooling2D(2,2),
        tf.keras.layers.Conv2D(64, (3,3), activation="relu"),
        tf.keras.layers.MaxPooling2D(2,2),
        tf.keras.layers.Flatten(),
        tf.keras.layers.Dense(512, activation="relu"),
        tf.keras.layers.Dense(1, activation="sigmoid")
    ])

# Compile model
model.compile(loss="binary_crossentropy",
                  optimizer=RMSprop(learning_rate=0.001),
                  metrics=["accuracy"]
              )

# Run model
history = model.fit(train_generator,
                    steps_per_epoch=2, 
                    epochs=100,
                    verbose=1,
                    callbacks=[callbacks])

print(history.history["accuracy"][-1])
