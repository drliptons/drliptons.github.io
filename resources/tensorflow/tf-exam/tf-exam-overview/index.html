    Coursera

    Note

    As of February 2022, coursera has updated its content to matches with the latest TensorFlow version. Some notes in this document might not be aligned with the latest coursera course content.

    The Coursera DeepLearning.AI TensorFlow Developer Specialization is a 4 parts course. Normally it charges an enrollment fee as a monthly subscription. However, the course offers a 7 days trial period. It is possible to finish the entire lectures and exercises within that period. As following are the links to my personally taken notes.

    Part 1: Introduction to TensorFlow for AI, ML, and DL

    Learn TensorFlow basics within 4 exercises. It is distributed into 4 weeks but can be all done in 1 day. Read all here

    • Week 1 - A New Programming Paradigm: predict house pricing with a very simple TensorFlow model
    • Week 2 - Introduction to Computer Vision: image classifications basics
    • Week 3 - Enhancing Vision with Convolutional Neural Networks: image classification using Convolutions and Pooling, with early stopping callback
    • Week 4 - Using Real-world Images: image classification using Convolutions and Pooling, with early stopping callback

    Part 2: Convolutional Neural Networks in TensorFlow

    Read all here

    • Week 1 - Exploring a Larger Dataset: create a model for binary-class image classification model
    • Week 2 - Augmentation: A technique to avoid overfitting: use data augmentation on a binary-class image classification model
    • Week 3 - Transfer Learning: use transfer learning to create an ML model
    • Week 4 - Multiclass Classifications: train a multi-class classification model on a rock, paper, scissor dataset

    Part 3: Natural Language Processing in TensorFlow

    Read all here

    • Week 1 - Sentiment in text: learn the basics of NLP and how to convert text into numbers. Preparing data to feed into a Neural Network with TensorFlow.
    • Week 2 - Word Embeddings: learn to use word embeddings with Keras layers and start to train a neural network
    • Week 3 - Sequence models: use word's position to create a sequence model
    • Week 4 - Sequence models and literature: create a NLP model to create a poetry

    Part 4: Sequence, Time Series and Prediction

    Read all here

    • Week 1 - Sequences and Prediction: learn the basics about time series data, and how to prepare the data
    • Week 2 - Deep Neural Networks for Time Series: earn how to create a simple dense neural network models
    • Week 3 - Recurrent Neural Networks for Time Series: use RNN to train a model and making prediction
    • Week 4 - Real-world time series data: use real world data to train a model and make prediction
