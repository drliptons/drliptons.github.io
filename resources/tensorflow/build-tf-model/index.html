    Download Data

    Before we can download the data, we need to identify where our Data is coming from. The data may come from a single source such as TensorFor Dataset or from Kaggle. Sometimes we might drew data from various sources to have a complete set of data.

    TensorFlow Dataset

    TensorFlow has an extensive lists of Datasets available for users to enter. You may call an overview function of all available dataset by importing import tensorflow_datasets as tfds and calling the tfds.list_builders() function

    import tensorflow_datasets as tfds

tfds.list_builders()

    A full list of available datasets will be shown, as an example as following

    ['abstract_reasoning',
 'accentdb',
 'aeslc',
 'aflw2k3d',
 'ag_news_subset',
 'ai2_arc',
 'ai2_arc_with_ir',
 'amazon_us_reviews',
...]

    For more datasets visit: TensorFlow Datasets Overview

    To easily load the dataset we can use the tensorflow function tfds.load() and adding some argument when needed. Some argument might be;

    • split: to split the dataset into sets (train, ['train', 'test'], 'train[80%:]', ...) see more here
    • shuffle_files: (bool) to shuffle files between epochs
    • data_dir: (str) location to save the dataset (Default: ~/tensorflow_datasets/)
    • with_info: (bool) to return a metadata (return: tfds.cose.DatasetInfo)

    for example

    dataset = tfds.load('fashion_mnist', split=['train', 'test'], shuffle_files=True)
