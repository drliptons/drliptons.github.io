    JSON

    JavaScript Object Notation or JSON is a lightweight, text-based, language-independent data interchange format. JSON defines a small set of formatting rules for the portable representation of structured data. Since it is so lightweight, it becomes one of the most popular data files for storing and transferring data.

    JSON stands for JavaScript Object Notation, which is a subset of JavaScript programming language. JSON is easy for both humans and machines to create and understand. It is very popular for storing and sending information. In this notebook, we will explore most of the basic functionalities of the JSON library.

    Import json library

    json library will enable to work with JSON files and data.

    # Add dependency
import json

    Putting everything together

    Now, let's put everything that we have done together as our own superheroes class.

    class Hero:
  def __init__(self, name, gender, real_name):
    self.name = name
    self.gender = gender
    self.real_name = real_name

  def print_data(self):
    print(f"{self.name} is a {self.gender} and the real name is {self.real_name}")

  def export_to_json(self, file_name):
    person_dict = {'name': self.name, 'gender': self.gender, 'real_name': self.real_name}
    with open(file_name, 'w') as f:
      f.write(json.dumps(person_dict, indent=2))
    print("Export success")

  def load_from_json(self, file_name):
    with open(file_name, 'r') as f:
      data = json.loads(f.read())
    self.name = data['name']
    self.gender = data['gender']
    self.real_name = data['real_name']
    print("Load success")

  def update_real_name(self, name):
    self.real_name = name
    print(f"Real name is updated to {self.real_name}")

  def delete_real_name(self):
    self.real_name = None
